# FeFET Adaptive Testing Framework

This project implements a cross-layer framework for degradation-aware fault modeling and adaptive test generation in FeFET memories.

## Pipeline
- TCAD → ΔVth extraction
- Empirical modeling
- RTL fault injection
- ML-based adaptive testing
